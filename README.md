# Credit Card Fraud Detection using Machine Learning

## 📌 Project Overview
This project aims to detect fraudulent credit card transactions using Machine Learning techniques.

The dataset is highly imbalanced, with very few fraud transactions compared to genuine transactions.

---

## 🚀 Features

- Exploratory Data Analysis (EDA)
- Class Imbalance Analysis
- Logistic Regression
- Random Forest Classifier
- SMOTE for balancing data
- Feature Importance Analysis
- Streamlit Web Application

---

## 📊 Dataset Information

- Rows: 284807
- Columns: 31
- Target Variable: Class
    - 0 → Genuine Transaction
    - 1 → Fraud Transaction

---

## 🤖 Models Used

1. Logistic Regression
2. Random Forest
3. Random Forest + SMOTE

---

## 📈 Results

| Model | Recall | F1 Score |
|------|---------|---------|
| Logistic Regression | 0.63 | 0.72 |
| Random Forest | 0.80 | 0.86 |
| Random Forest + SMOTE | 0.81 | 0.82 |
